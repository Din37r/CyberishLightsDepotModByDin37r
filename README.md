## About

This mod adds compatibility between Cyberish Lights and Dye Depot.

**Notice!** The textures for orange, lime, green, and cyan lights have been changed. Their old textures are used by some Dye Depot colored lamps. This is because the lamp colors in Cyberish Lights sometimes differ from the dye colors in order to match the majority of blocks in the game.
